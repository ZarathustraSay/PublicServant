## 格式规范

| 标题 |                       关于...调研报告                        |
| :--: | :----------------------------------------------------------: |
| 称谓 |                              ❌️                               |
| 开头 | 背景、目的、意义、调查范围、调查对象、调查方式、宏观问题/对策... |
| 主体 |                        找问题，找对策                        |
| 结尾 |                   可有可无，字数不够可不管                   |
| 落款 |                              ❌️                               |

开头的内容并非只有那些，根据申论材料适当增删。
